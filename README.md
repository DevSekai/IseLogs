# IseLogs
How to use IseLogs ?

You just need use this Trigger :

		from client side :
		TriggerServerEvent('Ise_Logs', Color, Title, Description)
		
		
		from server side :
		TriggerEvent('Ise_Logs', Color, Title, Description)
